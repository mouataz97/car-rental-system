# Car Rental System

Project scaffold for a simple car rental system.

Structure:

- config/: configuration files
- public/: publicly accessible PHP pages
- assets/: css, js, images
- includes/: shared headers, footers, auth
- uploads/: uploaded images
- sql/: database schema

How to use:
- Place this project in your web server root or configure virtual host.
- Import `sql/database.sql` into your MySQL/MariaDB server.
- Update `config/database.php` with real DB credentials.
